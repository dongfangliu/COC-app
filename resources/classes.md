## 生成人物卡

+ 形式：roll点 （(xDx+y)*z）或者购点
+ 基本属性导出战斗属性

## 游戏系统

+ 检定（暗骰，明骰，发布检定）
+ 资产变化、战斗属性变化、状态变化、
+ 战斗

##类

### 用户信息

+  id，昵称等其他基本用户信息
+ 我的团
+ 我的模组
+ 我的规则书
+ 我的经历
+ 上次登录时间
+ 变声模版

###人物卡

+ 所属团id
+ 所属规则书id
+ 见人物卡excel及playercard.png
+ 人物状态：已死亡，健康，临时疯狂等等

###团

+ 团id
+ 团状态：招募中，已满，已开，已结束
+ 是否可旁观
+ KP信息（用户公开信息）
+ 模组
+ 描述和KP备注
  + 一般为建卡限制
+ 参与人员信息
+ 上次活动时间
+ 历史记录-》仅对参与人员开放-》需要进入第三层界面
+ 结团评价
